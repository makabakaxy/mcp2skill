# Trending Commands

## list — List trending topics by frequency

```bash
mcp2cli trendradar trending list
mcp2cli trendradar trending list --top-n 20 --extract-mode auto_extract
mcp2cli trendradar trending list --mode daily --top-n 10
```

Also supports: `--top-n`, `--mode` (daily/current), `--extract-mode` (keywords/auto_extract)

Use `mcp2cli trendradar trending list --help` for full parameter details.
