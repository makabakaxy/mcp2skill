# Report Commands

## generate — Generate daily/weekly summary report

```bash
mcp2cli trendradar report generate --report-type daily
mcp2cli trendradar report generate --report-type weekly
mcp2cli trendradar report generate --report-type daily --date-range '{"start":"2025-01-01","end":"2025-01-01"}'
```

Also supports: `--report-type` (daily/weekly), `--date-range`

Use `mcp2cli trendradar report generate --help` for full parameter details.
