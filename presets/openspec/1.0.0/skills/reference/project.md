# Project Commands

## init — Initialize OpenSpec in your project

```bash
mcp2cli openspec project init
```

No parameters required.

## update — Update OpenSpec instruction files

```bash
mcp2cli openspec project update
```

No parameters required.

## refresh-cache — Force refresh the cached directory listing

```bash
mcp2cli openspec project refresh-cache
```

No parameters required. Use if you suspect changes were made outside OpenSpec tools.

Use `mcp2cli openspec project <cmd> --help` for full parameter details.
