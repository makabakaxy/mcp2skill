# Page Commands

## snapshot — Capture accessibility snapshot of the current page

```bash
mcp2cli browser page snapshot
```

> Use this to get element references (`ref`) before clicking, typing, or hovering.

## screenshot — Take a screenshot of the current page

```bash
mcp2cli browser page screenshot
```

## console-logs — Get console logs from the browser

```bash
mcp2cli browser page console-logs
```

## wait — Wait for a specified time in seconds

```bash
mcp2cli browser page wait --time 2
mcp2cli browser page wait --time 5
```

Use `mcp2cli browser page <action> --help` for full parameter details.
