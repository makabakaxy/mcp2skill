# Navigate Commands

## go — Navigate to a URL

```bash
mcp2cli browser navigate go --url https://example.com
mcp2cli browser navigate go --url https://github.com/login
```

## back — Go back to the previous page

```bash
mcp2cli browser navigate back
```

## forward — Go forward to the next page

```bash
mcp2cli browser navigate forward
```

Use `mcp2cli browser navigate <action> --help` for full parameter details.
